Ce projet a pour objectif d'étudier un jeu de données issu d'une source ouverte (Kaggle, Insee, data.gouv, etc.) afin de définir une ou plusieurs grandeurs statistiques d’intérêt et de construire des estimateurs permettant d’analyser leurs propriétés. Le travail consiste à explorer les données, proposer un ou deux estimateurs pertinents, étudier leurs performances empiriques (biais, variance, RMSE, intervalles de confiance) et produire des visualisations pour appuyer l’analyse.
